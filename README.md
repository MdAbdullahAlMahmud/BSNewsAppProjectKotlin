# A project to showcase current news from third party API.  

Feature List : 
 
* Splash Screen 
* OnBoard Screen 
* Breaking News
* Search News
* Add to favourite News
* News details 
* oAuth login



Technology/Concept/Stack Used

* Kotlin
* MVVM Design Pattern
* DI
* Navigation Component
* Single Activity App
* Room Database
* View Binding
* Coroutine Implementaion
* Unit Test
